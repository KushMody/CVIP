# CVIP Project

Computer Vision and Image Processing project implementation.

## Contents
- `blur.py` - Image blur functionality
- `det_track.py` - Detection and tracking
- `face_smile.py` - Face and smile detection
- `main.py` - Main application
- `tp.py` - Utility functions
- `tp.html` - HTML output

## YOLOv5 Model
Uses `yolov5s.pt` pre-trained model for object detection.
